# 📱 Guide d'Installation - Application CCNS

## Pour les Clients

### ⚠️ Avant de commencer

Cette application nécessite **Android 5.0 ou supérieur**.

---

## 📥 Étape 1 : Télécharger l'application

Vous avez reçu le fichier `app-arm64-v8a-release.apk` par :
- 💬 WhatsApp / Telegram
- 📧 Email
- 🔗 Lien Google Drive
- 🌐 Lien GitHub

**Téléchargez le fichier APK sur votre téléphone.**

---

## 🔧 Étape 2 : Autoriser l'installation

### Sur Android 8.0 et supérieur :

1. Ouvrir le fichier APK téléchargé
2. Un message apparaît : **"Pour votre sécurité, votre téléphone n'est pas autorisé à installer..."**
3. Cliquer sur **"Paramètres"**
4. Activer **"Autoriser cette source"** ou **"Installer des applications inconnues"**
5. Revenir en arrière avec le bouton retour

### Sur Android 7.1 et inférieur :

1. Aller dans **Paramètres** → **Sécurité**
2. Activer **"Sources inconnues"**
3. Confirmer en cliquant **"OK"**

---

## ✅ Étape 3 : Installer l'application

1. Rouvrir le fichier APK téléchargé
2. Cliquer sur **"Installer"**
3. Attendre la fin de l'installation (10-30 secondes)
4. Cliquer sur **"Ouvrir"** ou **"Terminé"**

✅ **L'application CCNS est maintenant installée !**

Vous la trouverez dans vos applications avec les autres apps.

---

---

## 🔄 Étape 5 : Utiliser l'application

### Tableau de bord

Vous voyez :
- 💰 **Dette Totale** : Total des crédits non payés
- 👥 **Clients Actifs** : Nombre de clients avec dette
- 📊 **Statistiques du mois** : Crédits et paiements du mois

### Gérer les clients

1. Cliquer sur **"Clients"** (icône personnes)
2. Voir la liste de vos clients
3. **Ajouter un client** :
   - Cliquer sur le bouton **"+"** en bas à droite
   - Remplir les informations
   - Cliquer **"Enregistrer"**
4. **Voir détails d'un client** :
   - Cliquer sur un client dans la liste
   - Voir ses transactions et sa dette

### Ajouter une transaction

1. Aller dans les détails d'un client
2. Cliquer sur **"Nouvelle Transaction"**
3. Choisir le type :
   - **Crédit** : Le client achète à crédit (dette augmente)
   - **Paiement** : Le client paie (dette diminue)
4. Entrer le **montant** en Dinars
5. Ajouter une **note** (optionnel)
6. Cliquer **"Enregistrer"**

### Voir les transactions

1. Cliquer sur **"Transactions"** (icône liste)
2. Voir toutes les transactions récentes
3. Filtrer par client si besoin

### Modifier votre profil

1. Cliquer sur **"Profile"** en haut à droite
2. Modifier vos informations
3. Cliquer **"Enregistrer"**

## ❓ Questions Fréquentes

### Q : L'installation est-elle sécurisée ?

**R :** Oui ! L'avertissement "Sources inconnues" apparaît simplement parce que l'app n'est pas encore sur Google Play Store. L'application est 100% sûre.

### Q : Pourquoi l'app demande-t-elle l'accès à Internet ?

**R :** L'application synchronise vos données avec le serveur pour que vous puissiez y accéder depuis plusieurs appareils et pour sauvegarder vos informations.

### Q : Mes données sont-elles privées ?

**R :** Oui ! Seul vous pouvez voir vos clients et vos transactions. Chaque utilisateur a son propre espace sécurisé.

### Q : Que faire si j'oublie mon mot de passe ?

**R :** Contactez le support à [votre@email.com] pour réinitialiser votre mot de passe.

### Q : L'application fonctionne-t-elle hors ligne ?

**R :** Non, une connexion Internet est nécessaire pour utiliser l'application car les données sont stockées dans le cloud.

### Q : Puis-je utiliser l'app sur plusieurs téléphones ?

**R :** Oui ! Installez l'app sur chaque téléphone et connectez-vous avec le même email/mot de passe. Vos données seront synchronisées.

### Q : Comment mettre à jour l'application ?

**R :** Lorsqu'une nouvelle version est disponible, vous recevrez le nouveau fichier APK. Installez-le simplement par-dessus l'ancienne version (vos données seront conservées).

### Q : L'app consomme-t-elle beaucoup de données ?

**R :** Non, très peu. Environ 1-2 MB par mois d'utilisation normale.

## 🆘 Problèmes et Solutions

### ❌ "Impossible d'installer l'application"

**Solutions** :
1. Vérifier que vous avez autorisé les "Sources inconnues"
2. Vérifier que vous avez assez d'espace (minimum 50 MB)
3. Redémarrer le téléphone et réessayer

### ❌ "L'application se ferme immédiatement"

**Solutions** :
1. Vérifier votre connexion Internet
2. Désinstaller et réinstaller l'application
3. Redémarrer le téléphone

### ❌ "Impossible de se connecter"

**Solutions** :
1. Vérifier votre connexion Internet (WiFi ou données mobiles)
2. Vérifier que votre email et mot de passe sont corrects
3. Attendre 30 secondes et réessayer (le serveur peut être en démarrage)

### ❌ "Les données ne se chargent pas"

**Solutions** :
1. Tirer vers le bas pour rafraîchir
2. Fermer et rouvrir l'application
3. Vérifier votre connexion Internet
