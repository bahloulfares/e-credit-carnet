# Contribuer au Projet

Merci de votre intérêt pour contribuer à ProCreditApp!

## Flux de Contribution

1. **Fork** le dépôt
2. **Créez une branche** pour votre feature (`git checkout -b feature/amazing-feature`)
3. **Committez** vos changements (`git commit -m 'Add amazing feature'`)
4. **Pushez** vers la branche (`git push origin feature/amazing-feature`)
5. **Ouvrez une Pull Request**

## Standards de Code

### TypeScript & ESLint
```bash
npm run lint
npm run lint -- --fix
```

### Formatage
```bash
npm run format
```

### Tests
```bash
npm test
npm run test:coverage
```

Tous les tests doivent passer et la couverture doit être > 80%.

## Conventions

### Commits
```
feat: Add authentication endpoints
fix: Fix client deletion issue
docs: Update API documentation
refactor: Improve error handling
test: Add unit tests for AuthService
```

### Pull Request Template
```markdown
## Description
Brief description of changes

## Type of Change
- [ ] Bug fix
- [ ] New feature
- [ ] Breaking change
- [ ] Documentation update

## How to Test
Steps to verify the changes

## Checklist
- [ ] Code follows style guidelines
- [ ] All tests pass
- [ ] Documentation updated
- [ ] No new warnings generated
```

## Signaler des Bugs

Utilisez GitHub Issues avec le template:

```markdown
### Description
Clear description of the bug

### Steps to Reproduce
1. ...
2. ...

### Expected Behavior
What should happen

### Actual Behavior
What actually happens

### Environment
- OS: (Windows/Linux/Mac)
- Node version: 
- MySQL version:
```
